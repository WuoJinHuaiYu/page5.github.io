Code

林浔是一个程序员，在又一次熬夜肝代码后，他突然发现自己的身体里多了一个程序输入窗口。
他试着编了一个循环程序……

第二天，他的邻居老头上门了：“小浔，你小小年纪，竟然已经跨入了炼气期，不如来做我门下弟子如何？”
第三天，街上闲逛，一个人拦住了他：“少年，我看你骨骼清奇，天资卓绝，不如就此加入我的宗门如何？”
林浔：？？？

直到后来，他发现现代社会竟然卧虎藏龙，藏着许多的修真者。
只是……别人修真靠的是灵力，相当困难，而他修真靠的是编程，还能看见其他人体内里的程序？！

某元婴大佬：竖子！你怎能窥破我剑招的破绽？
林浔淡定收键盘：你程序有bug。
某渡劫大佬：……我不甘心！你这招叫什么？
林浔扶眼镜：蒙特卡洛算法。

——然后，他就这样逐渐成为了修真界的传奇人物……

别人修仙靠灵力
他修仙靠C语言，Java和Python
别人是剑修
他是键修,键盘的键

林浔飞快选取赤霄龙雀剑上那些看起来很关键的代码
在手机上打出来，给王安全发过去。
林算法：安全，看个代码。
过了大概十分钟，林浔的手机响了一声。
王安全：流批。
王安全：你哪拷来的？你不是在山上吗？
林算法：这是什么？
王安全：我看着是杀毒软件源代码，绝对不是市面上那些，哪弄来的？
林算法：很厉害？
王安全：这样跟你说吧，我得再修炼个十年二十年，还不一定写出来。这是谁家的东西？赶紧能买股权买股权，等它面世，咱们就发了。
林算法：不好说，他们修真界一个诛邪破魔、以杀止杀的神器上写的。


“身份的转换需要时间。”林浔道：“我现在就像一个刚刚知道矩阵的人，您不能指望我能破译格密码……”
说着说着，他感到这个对话的走向不对。
自己怎么就不自觉跟东君杠起来了？
下一秒，一件他没有想到的事情发生了——东君也和他杠上了。
东君：“我以为你的学习能力足够让你做到这一点。”
林浔：“您高估我了。”
东君看着他。
林浔眨了眨眼睛。
东君：“我可以给你三分钟时间适应。”
林浔：“这无异于要我三分钟破解NP完备问题。”
